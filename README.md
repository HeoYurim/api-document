# api-document

코틀린 기반 프로젝트 연동규격서 관리 어드민 웹

---

## 💻 프로젝트 소개

사내에서 운영 중인 여러 프로젝트의 연동 규격서(API, 데이터 포맷, 인증 방식 등)를  
직원별·프로젝트별로 한눈에 확인할 수 있도록 만든 **어드민 웹 서비스**입니다.

기존에 문서가 흩어져 있거나,  
담당자별로 관리되어 파악이 어려운 문제를 해결하는 것을 목표로 합니다.

## ⚙️ 개발 환경

- **Language** : Kotlin
- **JDK** : 17 (또는 사용 버전)
- **Framework** : Spring Boot (3.x)
- **Build Tool** : Gradle (Kotlin DSL)
- **Database** : MySQL / PostgreSQL / H2
- **ORM** : JPA (Hibernate) / QueryDSL
- **Template Engine** : Thymeleaf (또는 React / Vue)
- **IDE** : IntelliJ IDEA

---

## 🗂️ 주요 기능

### 🔐 관리자 기능
- 관리자 로그인 / 로그아웃
- 권한 기반 접근 제어 (Admin / User)

### 👨‍💼 직원 관리
- 직원 등록 / 수정 / 삭제
- 직원별 담당 프로젝트 매핑

### 📦 프로젝트 관리
- 프로젝트 등록 / 수정 / 삭제
- 프로젝트 상태 관리 (운영 / 종료 / 테스트)

### 🔗 연동 규격서 관리
- 프로젝트별 연동 규격서 등록
- API 명세 관리 (URL, Method, Header, Body, Response)
- 인증 방식 관리 (Token, Key, OAuth 등)
- 버전 관리 및 변경 이력 확인

### 🔍 조회 기능
- 직원별 사용 중인 프로젝트 조회
- 프로젝트별 연동 규격서 한눈에 보기
- 키워드 기반 검색 (프로젝트명, API명)
